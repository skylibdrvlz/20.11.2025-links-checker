Link Checker Service
Сервис для проверки доступности веб-ссылок с возможностью генерации PDF-отчётов.


Функциональность

- Проверка ссылок: Массовая проверка доступности веб-сайтов
- История проверок: Сохранение результатов с уникальными ID
- PDF-отчёты: Генерация отчётов по истории проверок
- Graceful shutdown: Корректное завершение работы без потери данных


API Endpoints

1. Проверка ссылок

POST /check-links
Content-Type: application/json

{
"links": ["google.com", "github.com", "example.com"]
}


Генерация PDF-отчёта

POST /generate-report
Content-Type: application/json

{
"links_list": [1, 2, 15]
}

Используйте --output filename.pdf для сохранения PDF


Дополнительно:
- Если при запросе /generate-report в "links_list" указать не существующие id, то сработает ошибка. Если при этом использовать --output, то в PDF-файле так же будет текст ошибки с неизвестными id
- Любой запрос к сайту, который длится больше 5 секунд, считается "not available"
- Вместо базы данных используется файл data.json, который создается/редактируется при каждой новой записи
